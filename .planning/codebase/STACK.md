# Tech Stack

**Last Updated:** 2026-04-26

## Core Technology
- **Language**: TypeScript
- **Frontend**: React (v19)
- **Backend**: Node.js / Express (v4.21.2)
- **Build Tool**: Vite (v6.2.0)

## Frontend Ecosystem
- **Styling**: Tailwind CSS (v4) with tw-animate-css, shadcn, clsx, tailwind-merge
- **Components**: @base-ui/react, lucide-react
- **Routing**: react-router-dom
- **3D Graphics**: three.js, @react-three/fiber, @react-three/drei
- **Animations**: motion

## Backend Ecosystem
- **Database**: MongoDB (via mongoose v8.13.2)
- **Authentication**: JWT (jsonwebtoken), bcryptjs
- **Email**: nodemailer
- **Environment**: dotenv

## Development Tools
- **Execution**: tsx for running backend typescript
- **Linting/Formatting**: tsc
